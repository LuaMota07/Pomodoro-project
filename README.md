# Pomodoro Project

# Description
-  A simple Pomodoro Timer web application to help improve focus and productivity.

# Purpose 
-  Practice HTML, CSS and Javascript
-  Apply time management concepts using the Pomodoro Technique.
-  Build a functional productiviy tool.

# Features
-  Start and stop the timer
-  Countdown for work session
-  Break time control
-  Clean and easy-to-use interface 

# Technologies used
-  HTML
-  CSS
-  Javascript

# Project Structure
-  index.html - application structure
-  style.css - styling and layout
-  script.js - timer logic and behavior

# How to run
-  Clone the repository
-  Open index.html in your browser
-  Star a Pomodoro session

# Future improvements 
-  Sound notifications
-  Session Counter
